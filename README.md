# 📱 Mobile Portfolio | Bárbara Oliveira
Aplicação móvel desenvolvida com React Native e Expo para a disciplina de Dispositivos Móveis. O app funciona como um portfólio dinâmico, consumindo dados de uma API REST externa e apresentando uma interface moderna em tons de azul marinho.

## 🛠️ Tecnologias e Módulos
Framework: React Native + Expo (SDK 51)

Navegação: Expo Router (Tabs & Modals)

Consumo de API: Axios

Backend: Node.js + Express (Hospedado na Vercel)

Banco de Dados: PostgreSQL (NeonDB)

## 📱 Funcionalidades
Home: Apresentação pessoal e objetivo.

Experiência Profissional: Lista dinâmica consumida do banco de dados (ID 6).

Formação Académica: Detalhes da graduação na UNICAP e formação técnica.

Sobre: Listagem das tecnologias utilizadas no desenvolvimento.

## 🌟 Funcionalidade Extra: Integração com Linking API para abertura direta de conversa no WhatsApp.

## 🚀 Como Executar o Projeto
Clonar o repositório:

Bash
git clone https://github.com/barbaraluizaoliveira/cv-mobile.git
Entrar na pasta e instalar dependências:

Bash
cd meu-portfolio
npm install
Iniciar o Expo:

Bash
npx expo start
Testar: Escaneie o QR Code com o app Expo Go no seu telemóvel (Android ou iOS).
